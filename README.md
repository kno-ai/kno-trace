# kno-trace

The control room for Claude Code.

See what Claude is doing — right now, prompt by prompt. Read-only. No interference.

## Install

*Coming soon — Homebrew tap and binary releases.*

## Usage

```bash
kno-trace                  # auto-open latest session for current project
kno-trace --pick           # browse all sessions across all projects
kno-trace --session <path> # open a specific JSONL session file
```
